# Metro Transit (metro-transit)
Metro Transit provides a real-time departure information API for the Minneapolis-Saint Paul metropolitan transit system. The API supports creating transit departure displays and accessing real-time bus and train arrival data.

## APIs

### Metro Transit NexTrip API
The Metro Transit API provides real-time departure information for creating transit departure displays. It offers endpoints for routes, stops, and real-time departure data for the Twin Cities metro area.

**Human URL:** [https://svc.metrotransit.org/](https://svc.metrotransit.org/)
